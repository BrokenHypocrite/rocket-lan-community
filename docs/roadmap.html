Roadmap

Rocket LAN is in pre-release. This page is the source of truth for what's in the box, what's actively being worked on, and what's on the bench. It's hand-maintained; expect it to lag the latest build by a release or two at the edges.

Pre-release reality check. Things will move. Items here are descriptions of intent, not commitments to ship. The "planned" column especially is a snapshot of current thinking, not a contract.

Now shipping

Core capabilities that early-access builds already have.

Core session flow

  • Host & join sessions via Steam friends, with open/invite-only visibility.
  • Mesh peer-to-peer over QUIC with encrypted relay fallback.
  • Reconnect resilience — dropped peers retry for ~30s before being removed.
  • Steam rich presence so friends see and join you natively.

Custom maps

  • Bring-your-own import from disk, plus Workshop subscriptions.
  • BakkesPlugins community map browser built in.
  • Map sync with BLAKE3 hashing — local matches skip the download.
  • Mid-session map swaps propagate the new map identity (name + preview) to peers immediately.
  • Map Maker mode with disk-watch hot swap, resumes across app restarts.

The app

  • EAC-compatible launch path; works on Steam and Epic copies of Rocket League.
  • In-game overlay with roster, current map, and host-only swap control.
  • Custom themes with editor, presets, and export/import.
  • Install diagnostic for non-standard layouts.

In progress

Currently being built or polished for an upcoming early-access drop.

Reliability

  • Connection edge-case hardening — long-tail NAT/firewall pairings where the relay fallback is slow to engage.
  • Map sync resume for partially-downloaded bundles after a dropped peer.

Onboarding

  • First-launch tour for the host/join/invite flow.
  • Norton / McAfee guidance in the install diagnostic when third-party firewalls block discovery.

Planned

On the bench — order is not strict, and any of these can move.

Sessions

  • Match recording integration — generate replay metadata alongside the match files custom-map sessions produce.
  • Host-side voting for the next map.
  • Spectator mode for a player who joins after a match has started.

Maps

  • Map collections — curated sets you can swap between or share.
  • Underpass slot as a second host-controlled custom-map slot.

Quality of life

  • Tournament helpers — bracket / round timers integrated into the overlay.
  • Cross-region friend hinting when the relay path is the only option.

Not planned

  • Any feature that enables custom maps or mods in official online matchmaking. That's a line we don't cross.
  • Anything that injects into Rocket League's process or interferes with EAC. By design.
  • Telemetry or analytics. No.

Have something you'd want to see that isn't here? Open a feature request or float it as an idea.
